**Finora AI** is a state-of-the-art, AI-powered financial management dashboard designed for the future of personal finance (2026). It bridges the critical gap between traditional money management and medical preparedness, ensuring users are not just wealthy, but protected.

> [!TIP]
> **Hackathon Jury Note**: Finora AI uniquely integrates a **Healthcare Risk Engine** that correlates lifestyle habits with financial targets—a feature missing in 99% of current fintech apps.

---

## 🌟 Key Pillars

### 1. **The Intelligence Core 🧠**
- **AI Spending Personality**: Leveraging LLMs to analyze transaction history and define your financial character.
- **Smart Analytics**: Real-time spending breakdowns and automated budgeting limits.
- **AI Forecasting**: Predictive modeling for future balances and potential protection gaps.

### 2. **Health-Wealth Nexus 🏥**
- **Personalized Healthcare Fund**: Calculates optimal medical savings based on your Age, Lifestyle, and Smoking status.
- **Insurance Advisor 2026**: A premium dashboard featuring high-trust Indian insurers (Bajaj Allianz, HDFC ERGO, Care) with real-time **CSR (Claim Settlement Ratio)** transparency.

### 3. **The Growth Engine 📈**
- **Investment Hub**: Live stock simulation with risk-tailored suggestions (Safe, Moderate, Aggressive).
- **Profit Simulator**: Real-time tools to visualize how your wealth can grow over time.

## 🚀 Tech Stack & AI Architecture

### **Core Stack**
- **Frontend**: **React 19** + **Vite** (Ultra-fast HMR and build times)
- **Backend-as-a-Service**: **Supabase** (Real-time PostgreSQL, Secure Authentication, and RLS)
- **Styling**: **Scoped Vanilla CSS** (Custom "WOW UI" design system with glassmorphism and pastel variables)
- **Routing**: **React Router 7**

### **The AI Brain 🧠**
Finora AI leverages state-of-the-art NLP and predictive models via the **Hugging Face Inference API** to deliver personalized financial coaching:
- **Primary Model**: `google/flan-t5-base` — Used for text-to-insight generation for the Insurance Advisor and Spending Personality.
- **Fallback Logic**: Advanced local rule-based engines for insurance scoring (CSR, Network, Premium math).
- **Interactive Visualization**: Custom-built **SVG & Canvas** engines for real-time risk progress bars and investment profit simulators.

## 🏁 Getting Started

### Prerequisites
- Node.js (v18+)
- Supabase Account (for Database & Auth)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/nidhi-girubakharan/Finora-AI.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up environment variables:
   Create a `.env` file with your `VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY`.
4. Start the development server:
   ```bash
   npm run dev
   ```
